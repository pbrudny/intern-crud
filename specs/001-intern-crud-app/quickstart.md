# Quickstart Guide: Intern Profile Management System

**Date**: 2026-03-13  
**Feature**: 001-intern-crud-app

## Overview

This guide helps developers quickly set up and run the intern-crud web application locally.

## Prerequisites

- Python 3.14 installed
- uv (modern Python package manager) - Install with: `curl -LsSf https://astral.sh/uv/install.sh | sh`
- Git
- Text editor or IDE

## Quick Setup (2 minutes)

### 1. Clone and Navigate

```bash
# Clone the repository
git clone git@github.com:pbrudny/intern-crud.git
cd intern-crud

# Or if already cloned, navigate to it
cd /Users/piotrbrudny/pythony/intern-crud
```

### 2. Install Dependencies with UV

```bash
# UV automatically creates venv and installs dependencies
# No need to activate - uv run handles it automatically!
uv sync
```

That's it! UV automatically:
- Creates a virtual environment (`.venv/`)
- Installs all dependencies from `pyproject.toml`
- Creates a lockfile (`uv.lock`) for reproducible builds

### 3. Set Up Environment Variables

Create a `.env` file in the project root:

```bash
# .env
SECRET_KEY=your-secret-key-here-change-in-production
DATABASE_URL=sqlite:///intern_crud.db
FLASK_APP=run.py
FLASK_ENV=development
UPLOAD_FOLDER=uploads/resumes
MAX_CONTENT_LENGTH=2097152
```

### 4. Initialize Database

```bash
# Use 'uv run' to execute commands in the virtual environment
# No need to activate!
uv run flask db init
uv run flask db migrate -m "Initial migration"
uv run flask db upgrade
```

### 5. Run the Application

```bash
# Start development server with uv run
uv run flask run

# Or use the run script
uv run python run.py
```

The application will be available at: `http://localhost:5000`

## First-Time Usage

### 1. Register First User (Auto-Admin)

1. Navigate to `http://localhost:5000`
2. Click "Register"
3. Enter email and password
4. Submit form
5. **You are now an administrator** (first user automatically gets admin role)

### 2. Create Your Profile

1. After registration, you'll be redirected to login
2. Log in with your credentials
3. Click "Create Profile"
4. Fill in required fields:
   - Name
   - Email
   - Student Index Number (min 4 characters, alphanumeric)
   - Institution
   - Field of Study
   - Graduation Date
5. Optionally add:
   - Work Experience
   - Projects
   - LinkedIn URL
   - Availability
   - Preferred Locations
   - Resume (PDF, max 2MB)
6. Click "Submit"

### 3. View Profiles

1. Navigate to "Profiles" in the menu
2. See list of all student profiles
3. Click on any profile to view details

### 4. Admin Functions

As the first user (admin), you can:
- Edit any profile
- Delete any profile
- Promote other users to admin
- Revoke admin role from users

## Common Tasks

### Run Tests

```bash
# Run all tests with uv run
uv run pytest

# Run with coverage
uv run pytest --cov=app tests/

# Run specific test file
uv run pytest tests/unit/test_models.py

# Run with verbose output
uv run pytest -v
```

### Database Migrations

```bash
# Create new migration after model changes
uv run flask db migrate -m "Description of changes"

# Apply migrations
uv run flask db upgrade

# Rollback migration
uv run flask db downgrade

# View migration history
uv run flask db history
```

### Create Test Data

```bash
# Run seed script (to be created)
uv run python scripts/seed_data.py
```

### Add New Dependencies

```bash
# Add a new package
uv add package-name

# Add a development dependency
uv add --dev package-name

# Remove a package
uv remove package-name

# Update all dependencies
uv sync
```

### Clear Database

```bash
# Delete database file
rm intern_crud.db

# Re-initialize
flask db upgrade
```

## Troubleshooting

### Port Already in Use

```bash
# Use different port
uv run flask run --port 5001
```

### Database Locked

```bash
# Stop all Flask processes
# Delete database file
rm intern_crud.db
# Re-initialize
uv run flask db upgrade
```

### Import Errors

```bash
# Resync dependencies
uv sync

# Or reinstall everything
rm -rf .venv uv.lock
uv sync
```

### UV Not Found

```bash
# Install UV
curl -LsSf https://astral.sh/uv/install.sh | sh

# Or on macOS with Homebrew
brew install uv

# Verify installation
uv --version
```

### Dependencies Out of Sync

```bash
# Sync dependencies from pyproject.toml
uv sync

# Update lockfile
uv lock

# Update specific package
uv add package-name@latest
```

### File Upload Errors

```bash
# Ensure upload directory exists
mkdir -p uploads/resumes

# Check permissions
chmod 755 uploads
```
